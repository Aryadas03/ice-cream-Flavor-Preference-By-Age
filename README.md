## Project Overview
This project analyzes ice cream flavor preferences across different age groups using survey data.
The objective is to identify which flavors are most popular among Kids, Teens, Adults and Seniors.

The analysis follows an end-to-end data analytics workflow, including data preparation,EDA, SQL analysis and interactive dashboard creation.

---

- **Excel** ‚Äì Data cleaning and pivot table analysis
- **Python** - Data analysis and visualization using Pandas, Matplotlib and Seaborn 
- **SQL (MySQL)** ‚Äì Data aggregation and age group analysis  
- **Power BI** ‚Äì Interactive dashboard and data visualization  

---

## Dataset Description
- Total records: **102**
- Columns:
  - Age
  - Flavor
  - Age Group (Kids, Teens, Adults, Seniors)

The dataset represents survey responses collected to understand ice cream consumption preferences.

---

## Analysis Performed
- Grouped ages into meaningful age categories
- Counted flavor preferences by age group using SQL
- Built an interactive dashboard with filters and KPIs

---

## Key Insights
- Chocolate is the most popular flavor overall
- Kids show a strong preference for Chocolate and Vanilla
- Adults have more balanced flavor choices
- Vanilla is consistently popular across all age groups
